# Vercelで公開する手順

## 1. GitHubにアップロード

`broadcast-equipment-vercel` フォルダの中身をGitHubリポジトリに入れます。

## 2. Vercelでプロジェクト作成

1. Vercelにログインします。
2. `Add New...` → `Project` を押します。
3. GitHubリポジトリを選びます。
4. そのまま `Deploy` します。

## 3. Redis連携を追加

1. Vercelのプロジェクト画面を開きます。
2. `Marketplace` または `Storage` から Redis 連携を探します。
3. Redis provider を選んで作成します。
4. このプロジェクトに接続します。
5. 環境変数が自動で入ったら、もう一度デプロイします。

## 4. スマホで開く

VercelのURLをスマホで開きます。
ブックマーク、またはホーム画面に追加できます。

## 5. 限定公開について

URLを知っている人だけに共有する運用が一番簡単です。
より強く制限したい場合は、管理者合言葉やログイン機能を追加します。
